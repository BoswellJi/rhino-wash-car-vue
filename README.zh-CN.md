## 项目简介

小犀牛自助洗车以构筑“智慧便利生活”为使命，秉承“智慧便民，公益共享”的服务原则，提供一种新的洗车方式，彻底告别洗车难、洗车贵。
通过更快速、便捷、便宜的洗车方式，让人们释放出更多的时间和空间，享受动手洗车的乐趣，洗出健康生活新潮流！

## 自助洗车特性

**智能化服务**：物联网技术：通过STM32主控芯片或ESP32模块实现设备自动化控制，支持水流量监测、清洁剂余量实时监控 。

**24小时运营**：无人值守：通过小程序或自助终端完成支付、启动洗车，无需人工干预 。 ‌

**标准化流程**：固定清洗步骤：清水冲洗→泡沫清洁→吸尘处理等标准化操作，确保服务一致性 。

**节能环保**：循环水系统：减少单次洗车用水量60% 。 ‌

**数据管理**：远程监控：云服务器实时上传设备状态，支持故障预警。



## 技术栈

**Vue3**：前端采用Vue、Element UI

**Java**：后端采用Spring Boot、Spring Security、Redis。

**Jwt**：权限认证使用Jwt，支持多终端认证系统。
